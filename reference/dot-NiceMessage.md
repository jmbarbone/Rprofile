# Adds a nice message to the start

Adds a nice message to the start

## Usage

``` r
.NiceMessage(x = 1:2, space = TRUE)
```

## Arguments

- x:

  A vector of integers to select from. If the package isn't available
  another random message is generated – or not

- space:

  if `TRUE` adds another `"\n"` to the output
