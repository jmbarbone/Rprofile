# Search print

A more colorful way of printing
[`search()`](https://rdrr.io/r/base/search.html)

## Usage

``` r
.Search(pattern = ".")
```

## Arguments

- pattern:

  A regular expression to search for in the search path

## Value

Nothing, called for its side-effects
